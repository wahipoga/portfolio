# **Super Square Runner**
An addictive infinite runner inspired by the Chrome dinosaur, with a modern twist! Control a square dinosaur, dodge obstacles, collect special power-ups, and survive as long as you can in a world that gets faster and more challenging.

This is my very first game using **Python and Raylib**!

## **HOW TO PLAY**

1. **Jump** (Space / Up Arrow) over red cacti
2. **Duck** (Down Arrow) to avoid low-flying blue birds
3. **Collect** green power-ups for special abilities
4. **Shoot** (F) when you have the weapon to destroy obstacles
5. **Survive** as long as possible!

---

## **CONTROLS**

* **SPACE / UP ARROW:** Jump
* **DOWN ARROW:** Duck
* **F:** Shoot (if you have the weapon)
* **ESC:** Return to menu
* **R / ENTER:** Restart

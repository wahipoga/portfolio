HOW TO PLAY:
1. JUMP (Space/Up Arrow) over red cacti
2. DUCK (Down Arrow) to pass under low-flying blue birds
3. COLLECT green buffs to gain special powers
4. SHOOT (F Key) when you have the weapon to destroy obstacles
5. SURVIVE as long as possible!

CONTROLS:
- SPACE / UP ARROW: Jump
- DOWN ARROW: Duck (essential for low birds)
- F: Shoot (when you have the weapon)
- ESC: Return to menu
- R / ENTER: Restart
